# Trail Tracking App

## Overview
A mobile trail tracking application designed to help users record outdoor routes, view trip details, and interact with location-based trail information.

## Problem
Outdoor and trail users need a simple way to track routes, organize activity details, and view navigation-related information from a mobile device.

## My Role
I worked on the mobile app structure, interface design, and location-focused user flow. The project helped me strengthen my ability to build mobile applications around real-world navigation and activity-tracking use cases.

## Technologies
- Android Studio
- Java / Kotlin
- Location Services
- Maps / route-based UI concepts
- Mobile UI design

## Key Features
- Trail or route tracking workflow
- Location-based app behavior
- User-friendly screens for activity/trip details
- Mobile-first layout for outdoor use cases
