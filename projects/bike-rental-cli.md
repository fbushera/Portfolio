# Bike Rental CLI System

## Overview
A Java command-line application connected to an Oracle SQL database for managing bike rentals, bike returns, searches, and active rental records.

## Problem
A rental system needs reliable database operations for viewing inventory, searching available bikes, creating rentals, returning bikes, and tracking active rentals.

## My Role
I built and tested a Java CLI workflow that connects to Oracle SQL through JDBC and supports practical rental operations.

## Technologies
- Java
- Oracle SQL
- JDBC
- SQLPlus
- Command-line interface design

## Key Features
- Database connection through JDBC
- SQL script execution and table setup
- Bike search by type and location
- Active rental lookup by bike ID or bike type
- Rent and return operations
- Formatted CLI output

## Skills Demonstrated
- Backend database integration
- SQL querying and updates
- Java CLI design
- JDBC driver/classpath troubleshooting
- Transaction-style workflow thinking
