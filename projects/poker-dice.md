# Poker Dice Multiplayer App

## Overview
A real-time multiplayer Android dice game supporting 2–3 players, timed rounds, bust tracking, and synchronized game state.

## Problem
Multiplayer games require all players to see consistent game state, even when users join, leave, restart, or time out. This project focused on handling those real-time synchronization challenges in a mobile app.

## My Role
I worked on Firebase-based multiplayer flow, player state tracking, timed rounds, restart behavior, and debugging multiplayer edge cases across multiple emulators.

## Technologies
- Java
- Android Studio
- Firebase Authentication
- Firebase Realtime Database
- Android lifecycle handling

## Key Features
- 2–3 player multiplayer sessions
- Realtime game-state synchronization
- Timed rounds
- Bust tracking
- Player join/leave handling
- Restart and post-game flow
