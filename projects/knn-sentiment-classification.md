# k-Nearest Neighbor Sentiment Classification Model

## Overview
A machine learning and NLP project using k-Nearest Neighbor sentiment classification with careful data cleaning, preprocessing, and model evaluation.

## Problem
Sentiment classification depends heavily on preprocessing choices. Removing the wrong words can damage meaning, especially when sentiment depends on negation words such as `not`, `never`, and `no`.

## My Role
I worked on the machine learning workflow, text preprocessing decisions, model testing, and analysis of how data-cleaning choices affected classification quality.
