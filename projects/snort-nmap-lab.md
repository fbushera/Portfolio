# Snort IDS + Nmap Detection Lab

## Overview
A cybersecurity and networking lab where Snort++ was configured on Kali Linux to detect Nmap scanning activity and network reconnaissance.

## Problem
Security teams need to detect suspicious scanning and reconnaissance before attackers move deeper into a network. This project demonstrated how IDS rules and packet captures can reveal scanning behavior.

## My Role
I configured Snort++, wrote and loaded custom rules, generated traffic with Nmap, and verified detections through alerts and Wireshark captures.

## Technologies
- Kali Linux
- Snort++
- Nmap
- Wireshark
- Linux networking

## Key Work Completed
- Ran Snort++ using custom local rules
- Generated TCP and UDP scan traffic with Nmap
- Tested SYN, TCP connect, UDP, and Xmas scan behavior
- Captured packets in Wireshark
- Compared IDS alerts with observed packet behavior
