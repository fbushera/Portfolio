# OS/161 System Calls Project

## Overview
A systems programming project focused on implementing core process-related system calls inside the OS/161 teaching kernel.

## Problem
Operating systems must manage process creation, execution, parent-child relationships, waiting, and process termination reliably. This project required implementing these behaviors at the kernel level.

## My Role
I worked on syscall implementation and debugging for process management features including `fork()`, `execv()`, `waitpid()`, `getpid()`, and `getppid()`.

## Technologies
- C
- OS/161
- System/161
- Linux command line
- GDB

## Key Work Completed
- Implemented process ID and parent process behavior
- Worked on process lifecycle management
- Implemented and tested syscall behavior
- Debugged kernel build and runtime issues
